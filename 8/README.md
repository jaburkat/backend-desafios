# Login por formulario

## Start

1. Configuración del archivo .env.example y luego renombrarlo:

- PORT
- URL MONGO LOCAL
- URL MONGO ATLAS
- NOMBRE DB
- NOMBRE SESSIONS
- COOKIE KEY
- SECRET KEY

   ```bash
   cp .env.example .env
   ```

2. Instalación de dependencias e iniciar servidor:
    ```shell
    npm install
    ```

    ```shell
    npm start
    ```
3. Ingresar a la APP:

    ```
    http://localhost:<PORT>
    ```
